api / routes.py

All HTTP routes mounted on the FastAPI app. Ollama-compatible plus Ghost-specific.

Auth

Every authed route validates the X-Ghost-Key header against request.app.state.args.api_key (which defaults to $GHOST_API_KEY; there is no hardcoded fallback key). An unset or empty key disables auth — allowed silently only on loopback binds. A non-loopback bind with an unset key refuses to boot (enforce_api_key_policy in main.py); an explicit --api-key '' boots with a loud warning.

CORS

Wildcard origin (allow_origins=["*"]) with credentials disabled. No cookies — auth is header-only.

POST /api/memory/correctX-Ghost-KeySurgically rewrite ONE vector-memory fragment: {"match": <exact text or unique substring>, "replacement": <new text>} → VectorMemory.correct_fragment (in-process — Chroma's persist dir must never be opened by a second process). Refuses ambiguous matches with 409.
POST /api/game/move · GET /api/game/gamesX-Ghost-KeyParticipant-mode turn-based games — the agent PLAYS (see game_routes.py, own router included before the catch-all). Game-agnostic: game selects an adapter (default chess; tictactoe ships as a second reference game), body {game?, state?, user_move?, history?} (fen is a backward-compatible alias for state). The adapter validates/applies the user's move and detects endings with CODE; the AGENT'S LLM then chooses the reply (MOVE:-line prompt over the legal list, ≤3 attempts with illegal-move feedback) and returns coaching text — critique (its read on the user's move), move_explanation (the idea behind its own move), and a friendly comment — for learning clients. 422 bad state / illegal or empty-string user_move (omission must be explicit — a bare Enter as "" once made the agent play both sides); 400 unknown game; 502 if no legal move after retries — deliberately no engine fallback (that would reintroduce the "random AI" the participant-mode rule forbids). GET /api/game/games lists the registered games. See the dedicated page for the adapter contract and per-game rules. ♟️ log line per move.
POST /api/uploadX-Ghost-KeyUpload a file into the sandbox. Path validated (no .., no absolute paths). 100 MB cap.
GET /api/download/{filename:path}X-Ghost-KeyStream a sandbox file. Path traversal guard.
{any other path}X-Ghost-KeyCatch-all proxy — forwards unknown requests upstream via agent.context.llm_client.http_client. Detects streaming from stream: true JSON body. Sets X-Accel-Buffering: no for SSE so nginx doesn't buffer.
